# Apex_Store_Data_Analysis

A comprehensive data analysis project performed using Excel to explore, clean, and visualize sales data for the Apex Store. This project highlights essential data analytics techniques such as data cleaning, transformation, and creation of interactive dashboards.

# Project Overview

The Apex Store Sales Analysis project aims to uncover meaningful insights from the store’s sales dataset. Using Excel, the dataset was cleaned, and analyzed to identify sales according to gender, states and platforms.

# Data Cleaning & Preparation

- Removed duplicates and blanks
- Standardized column names and formats
- Cleaned categorical fields
- Converted data types (dates, numbers,)
- Created derived columns for analysis (e.g., Age Group, Months)

# Key Analysis Performed

- Monthly & yearly sales performance
- Best-selling products
- High-revenue categories
- Low-performing items
- Orders by gender, age group, city

# Dashboard

An interactive Excel dashboard was created consisting of:

- Graphs and Charts (Sales vs Order, Order Status, Age vs Gender)
- Product category breakdown
- Sales by state/city
- Sales trend chart
- Slicers for dynamic filtering

# Tools & Technologies

- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers & Filters
